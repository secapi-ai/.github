# SEC API

SEC API is the SEC data API designed for agents.

It turns SEC filings and adjacent market data into traceable, source-backed investor workflows across filings, statements, ownership, governance, market data, events, webhooks, and agent-ready intelligence.

## For Agents

Use the documentation index and setup prompt when handing SEC API work to a coding agent:

- [`llms.txt`](https://docs.secapi.ai/llms.txt)
- [Give this prompt to your agent](https://docs.secapi.ai/give-this-prompt-to-your-agent)
- [LLM and agent guide](https://docs.secapi.ai/llm-guide)

Recommended first workflow: resolve one real company, run one filing or statements workflow, then preserve `requestId`, provenance, freshness, materialization metadata, and trace references in logs or user-facing output.
