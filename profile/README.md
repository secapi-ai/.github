# SEC API

SEC API gives software, researchers, and AI agents source-linked SEC filings, normalized company facts, ownership data, and search and market-data workflows.

## Start Here

1. [Create an API key](https://secapi.ai/signup), then follow the [first request flows](https://docs.secapi.ai/first-request-flows) to resolve a company and retrieve a filing.
2. Choose the interface that fits your work:
   - [REST API](https://docs.secapi.ai/api-reference) for direct HTTP control and custom integrations.
   - [SDKs](https://docs.secapi.ai/libraries-and-sdks) for JavaScript, Python, Go, or Rust applications.
   - [CLI](https://docs.secapi.ai/cli) for terminal-first exploration and automation.
   - [Hosted MCP](https://docs.secapi.ai/mcp-workflows) for MCP-aware agents and clients.
   - [Agent skills](https://github.com/secapi-ai/secapi-skills) for installable, repeatable research workflows.

## Common Workflows

- [Find the latest filing and extract a section](https://docs.secapi.ai/first-request-flows), such as 10-K risk factors.
- [Read normalized statements and company facts](https://docs.secapi.ai/api-reference/statements/get-v1-statements-all) with filing provenance.
- [Track institutional ownership changes](https://docs.secapi.ai/tutorials/monitor-13f-holdings) and [insider activity](https://docs.secapi.ai/api-reference/insiders/get-v1-insiders).
- [Search filing text](https://docs.secapi.ai/tutorials/semantic-search-risk-factors) for topics such as risk factors or disclosures.
- [Build a filing monitor](https://docs.secapi.ai/tutorials/build-filing-monitor) with delivery workflows.
